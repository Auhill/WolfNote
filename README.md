# 🐺 WolfNote - 狼人杀专用结构化笔记

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**WolfNote** 是一款专为狼人杀玩家设计的结构化笔记应用。它抛弃了传统笔记软件杂乱无章的记录方式，通过自定义的指令系统和多维度视图，帮助你清晰地梳理逻辑、追踪投票、管理身份分线。

> **移动端优先**：支持 PWA，可直接“安装”到手机桌面，获得原生 App 般的流畅体验。

---

## ✨ 核心特性

- 📝 **结构化输入**：采用类似快捷指令的标记语言，极速记录投票、发言和身份。
- 📅 **天数追踪**：自动记录每一条信息发生的日期，支持跨天查看对局复盘。
- 👤 **人员视图**：点击玩家头像即可查看其全程的“心路历程”——包括投给谁、被谁投、说过什么。
- 🔍 **身份分线 (Hypothesis)**：支持建立多个逻辑分支（如“1号是真预” vs “2号是真预”），在不同假设下推导狼坑。
- 🎨 **模糊匹配**：完美支持手机输入法，中英文冒号、逗号、顿号自动识别转换。
- 💀 **状态管理**：实时标记玩家的“倒钩”、“冲锋”状态及死亡/出局情况。

## 🚀 快速上手 (指令参考)

在主页输入框中直接输入以下指令即可完成记录：

| 指令 | 说明 | 示例 |
| :--- | :--- | :--- |
| `v[目标]:[投票者]` | 记录普通投票 | `v3:1,2` (1、2投给3) |
| `vs[目标]:[投票者]`| 记录警长投票 | `vs3:1,2` |
| `v0:[投票者]` | 记录弃票 | `v0:5` (5号弃票) |
| `b[玩家]:[身份]` | 标记身份(支持权重) | `b1:预言家0.8` |
| `s[玩家]:[状态]` | 标记逻辑状态 | `s1:倒钩` |
| `t[玩家]:[内容]` | 记录关键发言 | `t1:对话3号` |
| `d[玩家]` | 记录玩家死亡 | `d1` (1号死亡) |
| `o[玩家]` | 记录投票出局 | `o1` (1号出局) |

## 📦 部署与使用

1. **克隆项目**:
   ```bash
   git clone https://github.com/Auhill/WolfNote.git
   ```
2. **安装依赖**:
   ```bash
   npm install
   ```
3. **本地开发**:
   ```bash
   npm run dev
   ```
4. **构建上线**:
   ```bash
   npm run build
   ```
