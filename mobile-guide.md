# WolfNote 手机端使用指南

为了在手机端获得最佳体验（像 App 一样使用），你可以采用以下几种方式：

## 1. 局域网预览（快速测试）
如果你在电脑上运行开发服务器，且手机和电脑在同一个 Wi-Fi 下：
1. 在电脑终端运行：`npm run dev -- --host`
2. 终端会显示一个 `Network: http://192.168.x.x:5173/` 的地址。
3. 在手机浏览器（推荐 Chrome 或 Safari）中输入该地址。

## 2. 安装为 PWA（像 App 一样）
本应用已配置 PWA 支持。
- **iOS (Safari)**: 打开网页 -> 点击分享按钮 -> 选择“添加到主屏幕”。
- **Android (Chrome)**: 打开网页 -> 点击右上角三个点 -> 选择“安装应用”或“添加到主屏幕”。
安装后，应用将拥有独立的图标，且没有浏览器的地址栏，体验与原生 App 无异。

## 3. 部署到互联网（永久使用）
你可以将代码推送到 GitHub，然后使用以下服务进行免费托管，这样随时随地都能访问：
- **Vercel / Netlify**: 关联 GitHub 仓库后自动部署，支持 HTTPS。
- **GitHub Pages**: 在仓库设置中开启。

## 4. 指令快速参考
在主页输入框中：
- `v3:1,2` - 1,2号投3号
- `s1:倒钩` - 标记1号状态为“倒钩”
- `s2:冲锋` - 标记2号状态为“冲锋”
- `b1:预言家0.8` - 标记1号为预言家（80%权重）
- `t1:对话3号` - 记录1号发言
